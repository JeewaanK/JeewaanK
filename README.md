## Featured Projects

### Airport PageRank

Applied the PageRank algorithm to a U.S. airport route network to identify structurally important airports and simulate the effect of removing a major hub.

Key ideas used:

- Directed graph modeling
- Adjacency and transition matrices
- Markov chains
- Google Matrix with damping factor
- Power iteration and eigenvector centrality
- Hub-closure sensitivity analysis

Repository: https://github.com/JeewaanK/Airport_PageRank
